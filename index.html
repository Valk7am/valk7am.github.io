ABOUT ME

I'm Mahmoud Abuhassan, a Full Stack Developer originally from Palestine, currently based in Doha, Qatar, and working as a Software Engineer at Al Jazeera Media Network.

I work across the full development stack — from configuring Apache2 servers and managing backend systems, to crafting custom front-end experiences in WordPress and building backend services using Python with Flask. I've also developed complete projects using Flask, taking full ownership from backend to frontend.

I prefer using pure HTML, CSS, and JavaScript when possible, believing that the simplest solution is often the best. I'm also experienced with React, React Native, Expo, Bootstrap, and jQuery, and I'm currently exploring Next.js and Tailwind CSS to keep up with modern development trends.

On the backend, I'm well-established in API and REST API development, with hands-on experience in Java, PHP, MongoDB, Firebase, SQL, and PostgreSQL, all within Linux-based environments.

I've worked with headless CMS platforms like Contentful, and I'm currently exploring open-source solutions such as Ghost and similar systems to broaden my content architecture skills.

I'm a native Arabic speaker and fluent in English, bringing cultural awareness and strong communication to every team I join. Outside of work, I enjoy technical video games that challenge my creativity and logic — much like the systems I build every day.
